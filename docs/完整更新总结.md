## 🎯 设计特点总结

### 视觉风格
- 🌑 **暗黑主题** - 深灰/黑背景
- 💎 **玻璃态** - 毛玻璃 + 半透明
- 🌈 **渐变丰富** - 多彩渐变配色
- ✨ **光影效果** - 发光/聚光灯

### 动画系统
- ⚡ **Framer Motion** - 流畅动画
- 📜 **滚动触发** - 视差效果
- 🎯 **交互反馈** - 悬停/点击
- 🌊 **瀑布效果** - 交错进入

### 配色方案
```css
/* 背景 */
bg-gray-900/950 (深灰/黑)

/* 玻璃 */
bg-white/5 (半透明白)
border-white/10 (淡边框)
backdrop-blur-xl (强模糊)

/* 渐变 */
蓝色: from-blue-500 to-cyan-500
紫色: from-purple-500 to-pink-500
绿色: from-green-500 to-emerald-500
橙色: from-orange-500 to-red-500
黄色: from-yellow-500 to-amber-500
```

## 🎨 组件使用示例

### SpotlightCard
```tsx
<SpotlightCard 
  className="p-8"
  spotlightColor="rgba(168, 85, 247, 0.15)"
>
  {children}
</SpotlightCard>
```

### GlassCard
```tsx

  {children}
</GlassCard>
```

### GradientText

  Your Text
</GradientText>
