# 🌊 Floating Lines 背景替换说明

## ✅ 已完成的更新

成功将 About 区域的 **Silk 丝绸背景** 替换为 **Floating Lines 浮动线条背景**，并配置了明暗主题的动态渐变色！

---

## 🎨 Floating Lines 特性

### 视觉效果
- 🌊 **波浪线条**: 流畅的波浪形线条，优雅动感
- 🎨 **渐变色彩**: 支持多色渐变，视觉丰富
- ✨ **动态动画**: 线条自然流动，富有生命力
- 🖱️ **鼠标交互**: 鼠标移动时线条会弯曲响应
- 📐 **视差效果**: 鼠标移动产生视差，增加层次感

### 技术特点
- 基于 Three.js 渲染
- 支持多层波浪（上中下）
- 可配置线条数量、间距、速度
- 支持 CSS 混合模式

## 🔧 技术实现

### 代码结构

```tsx
import FloatingLines from './FloatingLines'
import { useTheme } from '../contexts/ThemeContext'

export default function About() {
  const { theme } = useTheme()
  
  // 根据主题配置渐变色
  const linesGradient = theme === 'light' 
    ? ['#ec4899', '#d946ef', '#c084fc', '#a78bfa', '#818cf8'] // 粉紫色渐变
    : ['#3b82f6', '#6366f1', '#8b5cf6', '#a855f7', '#d946ef'] // 蓝紫色渐变

  return (
    <div className="absolute inset-0 z-0">
      <FloatingLines 
        linesGradient={linesGradient}  // 动态渐变色
        lineCount={5}                  // 每层5条线
        lineDistance={5}               // 线条间距5
        interactive={true}             // 启用鼠标交互
        parallax={true}                // 启用视差效果
        animationSpeed={1}             // 动画速度1x
      />
    </div>
  )
}
```

### 当前配置

| 参数 | 值 | 说明 |
|------|-----|------|
| **linesGradient** | 动态数组 | 5色渐变，根据主题切换 |
| **lineCount** | 5 | 每层5条线，视觉平衡 |
| **lineDistance** | 5 | 线条间距适中 |
| **interactive** | true | 鼠标移动时线条弯曲 |
| **parallax** | true | 鼠标移动产生视差 |
| **animationSpeed** | 1 | 标准速度，流畅自然 |

## 🎯 完整参数说明

### 可选配置参数

```tsx
<FloatingLines 
  // 颜色配置
  linesGradient={['#color1', '#color2', ...]}  // 最多8个颜色
  
  // 波浪层配置
  enabledWaves={['top', 'middle', 'bottom']}   // 启用哪些层
  
  // 线条配置
  lineCount={5}                    // 或 [5, 6, 4] 为每层单独设置
  lineDistance={5}                 // 或 [5, 6, 4] 为每层单独设置
  
  // 位置配置
  topWavePosition={{ x: 0, y: 0, rotate: 0 }}
  middleWavePosition={{ x: 0, y: 0, rotate: 0 }}
  bottomWavePosition={{ x: 2, y: -0.7, rotate: -1 }}
  
  // 动画配置
  animationSpeed={1}               // 速度倍数
  
  // 交互配置
  interactive={true}               // 鼠标交互
  bendRadius={10}                  // 弯曲半径
  bendStrength={-5}                // 弯曲强度
  mouseDamping={0.05}              // 鼠标平滑
  
  // 视差配置
  parallax={true}                  // 视差效果
  parallaxStrength={0.2}           // 视差强度
  
  // 混合模式
  mixBlendMode="screen"            // CSS 混合模式
/>
```

## 🖱️ 交互特性

### 鼠标移动交互
- **弯曲效果**: 鼠标接近时线条会弯曲
- **跟随响应**: 线条实时跟随鼠标位置
- **平滑过渡**: 带阻尼的平滑动画

### 视差效果
- **层次感**: 不同层的线条移动速度不同
- **深度感**: 鼠标移动产生3D视差
- **沉浸感**: 增强用户互动体验

### 自动动画
- **流动效果**: 线条持续波浪流动
- **自然感**: 模拟水波的自然运动
- **循环动画**: 无缝循环，永不停止

## 🎨 渐变色设计理念

### 浅色模式：温暖渐变
```
粉红 → 玫瑰紫 → 亮紫 → 柔和紫 → 靛蓝
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
温暖 ←────────────────────→ 冷静
```

**设计意图**:
- 从温暖的粉色开始，给人亲切感
- 过渡到紫色，展现创造力
- 结束于蓝色，带来稳定感
- 整体温暖明亮，适合日间

### 暗色模式：冷艳渐变
```
蓝色 → 靛蓝 → 紫色 → 亮紫 → 玫瑰紫
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
冷静 ←────────────────────→ 热情
```

**设计意图**:
- 从冷静的蓝色开始，科技感
- 过渡到紫色，展现神秘感
- 结束于玫瑰紫，增加活力
- 整体冷艳深邃，适合夜间

## 🎨 自定义建议

### 调整线条数量
```tsx
// 更密集
lineCount={8}
lineDistance={3}

// 更稀疏
lineCount={3}
lineDistance={8}
```

### 调整动画速度
```tsx
// 更快
animationSpeed={2}

// 更慢
animationSpeed={0.5}
```

### 调整交互强度
```tsx
// 更强烈的弯曲
bendStrength={-10}
bendRadius={15}

// 更温和的弯曲
bendStrength={-2}
bendRadius={5}
```

### 禁用某些效果
```tsx
// 禁用交互
interactive={false}

// 禁用视差
parallax={false}

// 只显示顶层波浪
enabledWaves={['top']}
```

