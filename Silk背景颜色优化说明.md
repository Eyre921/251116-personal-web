## 🔧 技术实现

### 代码修改

**About.tsx**:
```tsx
import { useTheme } from '../contexts/ThemeContext'

export default function About() {
  const { theme } = useTheme()
  
  // 根据主题动态设置 Silk 颜色
  const silkColor = theme === 'light' ? '#d8b4fe' : '#6366f1'

  return (
    <section>
      <div className="absolute inset-0 z-0">
        <Silk 
          color={silkColor}        // 动态颜色
          speed={5}                 // 波动速度
          scale={1}                 // 缩放比例
          noiseIntensity={1.5}      // 噪点强度
        />
      </div>
    </section>
  )
}
```

### 关键点

1. **使用 ThemeContext**: 获取当前主题状态
2. **动态颜色选择**: 根据 theme 值切换颜色
3. **即时响应**: 主题切换时自动更新颜色
4. **完整配置**: 显式设置所有 Silk 参数

## 🎨 颜色选择理由

### 为什么选择紫色系？

#### 1. 品牌一致性
整站使用了紫色系作为主题色：
- Hero 区域：紫色渐变 (`from-purple-600 to-pink-600`)
- 导航栏：紫色点缀
- 按钮：紫色悬停效果

#### 2. 情感表达
- **紫色** 代表创造力、想象力、独特性
- **粉紫色** 增添温暖和亲和力
- **蓝紫色** 展现专业和科技感

#### 3. 用户体验
- **浅色**: 柔和不刺眼，长时间浏览舒适
- **暗色**: 神秘有质感，夜间浏览护眼

#### 4. 视觉层次
- 与其他背景（LiquidEther、Particles）形成差异
- 不同区块有独特的视觉身份
- 保持整体协调统一

## 🎯 Silk 参数说明

### 当前配置
```tsx
<Silk 
  color={silkColor}        // 主颜色（动态）
  speed={5}                // 波动速度（1-10，默认5）
  scale={1}                // 图案缩放（0.5-2，默认1）
  noiseIntensity={1.5}     // 噪点强度（0-3，默认1.5）
/>
```

### 参数详解

| 参数 | 当前值 | 范围 | 说明 |
|------|--------|------|------|
| **color** | 动态 | 任何 hex | 丝绸波纹的主色调 |
| **speed** | 5 | 1-10 | 波动速度，5为中等 |
| **scale** | 1 | 0.5-2 | 图案缩放，1为原始大小 |
| **noiseIntensity** | 1.5 | 0-3 | 噪点强度，1.5为中等 |
| **rotation** | 0 | 0-2π | 图案旋转角度（可选） |

## 🔍 微调建议

### 如果觉得太亮/太暗

调整遮罩透明度：
```tsx
// About.tsx 中的遮罩层
<div className="absolute inset-0 z-[1] 
  bg-gradient-to-b 
  from-white/50 via-gray-50/50 to-white/50      // 增加到50%
  dark:from-gray-900/50 dark:via-gray-950/50 dark:to-gray-950/50 
  pointer-events-none" 
/>
```

### 如果想要更强对比

使用更饱和的颜色：
```tsx
// 浅色模式：更深的紫色
const silkColor = theme === 'light' ? '#c084fc' : '#6366f1' // purple-400

// 暗色模式：更亮的紫色
const silkColor = theme === 'light' ? '#d8b4fe' : '#818cf8' // indigo-400
```

### 如果想要更柔和

降低噪点强度：
```tsx
<Silk 
  color={silkColor}
  noiseIntensity={1.0}  // 从1.5降到1.0
/>
```
